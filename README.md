# Prison-Management-System-



The Prison  Management System is a software application designed to automate and streamline the administrative tasks of a prison facility. It helps manage prisoner records, staff details, visitor information, and overall jail operations efficiently.

This system reduces manual work, improves accuracy, and ensures better organization of critical data.

🚀 Features
➕ Add Prisoners
Easily register and store details of new inmates.
🏢 Check Vacancy
View available space/capacity in the jail.
👮 Manage Staff
Keep track of available staff members and their details.
📋 View Inmates
Access and review complete prisoner records.
🔄 Transfer Prisoners
Transfer inmates between cells or facilities when needed.
📅 Schedule Visits
Manage and organize visitation schedules.
👤 Visitor Information Management
Store and maintain records of visitors ملاقات (visitors) for security and tracking.
🛠️ Technologies Used
Language: ( C#)
Framework: ( .NET, Windows Forms )
Database: ( SQL Server / MySQL)
🎯 Objectives
Automate prison  administration tasks
Maintain accurate and secure records
Improve efficiency and reduce paperwork
Provide quick access to critical information
